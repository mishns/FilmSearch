## Install

```sh
npm install
```

## Usage

You need to insert your kinopoisk token to constants.ts file to use the app. Otherwise, app can't get films info. You can get free token at https://api.kinopoisk.dev/documentation#/.


Dev mode:
```sh
npm run dev
```
Prod mode:
```sh
npm install -g serve
npm run build-prod
npx serve dist
```
